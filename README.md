# ExamAI – Intelligent Examination & Learning Platform
ExamAI is an AI-powered examination and learning platform that enables secure online assessments, AI-generated question papers, intelligent proctoring, automated evaluation, personalized study plans, AI viva practice, and performance analytics for students and educators.
Built with Next.js, Firebase, Firestore, Tailwind CSS, and Google Gemini AI, ExamAI enhances exam integrity while delivering personalized learning experiences through artificial intelligence.
